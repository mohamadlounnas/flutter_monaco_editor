# flutter_monaco_editor

Complete Flutter wrapper for Monaco Editor. See the [repository root](../../README.md) for overview and status.

## Installation

```yaml
dependencies:
  flutter_monaco_editor: ^0.1.0
```

Bundled Monaco version: **0.55.1**.
